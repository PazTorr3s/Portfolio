# Paz Torres – Developer Portfolio
### Personal Developer Portfolio built with React

Portfolio personal de desarrollo web creado con **React**, enfocado en mostrar proyectos, experiencia y formación académica de manera clara, moderna y responsive.

El sitio cuenta con **soporte multilenguaje (Español / Inglés)** y presenta proyectos reales desarrollados durante mi formación académica y práctica personal.

---

## Tecnologías Utilizadas

- **Frontend:** React, JavaScript, HTML, CSS
- **Estilos:** CSS Modules
- **Build Tool:** Vite
- **Iconos:** React Icons
- **Gestión de estado:** React Hooks, Context API
- **Control de versiones:** Git & GitHub
